# VoxScout Web

This is a Next.js project for VoxScout, built with React, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Font**: Inter (Google Fonts)
- **Animations**: Custom CSS mouse-tracking glow effects

## Project Structure

- `app/`: Application source code (pages, layout, styles)
- `components/`: Reusable UI components (AnimatedCard, etc.)
- `lib/`: Utility functions
