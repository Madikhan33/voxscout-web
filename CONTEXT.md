# VoxScout - Полный контекст проекта для ИИ

## Обзор проекта

**VoxScout** — это современная веб-платформа для проведения AI-интервью с кандидатами. Платформа предоставляет два основных портала: для HR-специалистов (работодателей) и для кандидатов (соискателей).

### Основная миссия
Автоматизация процесса первичного отбора кандидатов с помощью искусственного интеллекта, предоставляя объективную оценку навыков и экономя время HR-команд.

## Основные функции платформы

### Для HR-специалистов (Работодателей)

#### 1. **Создание и управление интервью**
- Создание шаблонов интервью с кастомными вопросами
- Настройка критериев оценки
- Управление активными и архивными интервью
- Отслеживание количества вопросов и кандидатов

#### 2. **Приглашение кандидатов**
- Генерация уникальных ссылок для прохождения интервью
- Отправка приглашений кандидатам
- Отслеживание статуса прохождения (приглашен, начал, завершил)

#### 3. **Аналитика и отчеты**
- Дашборд с ключевыми метриками:
  - Активные интервью
  - Приглашенные кандидаты
  - Завершенные интервью
  - Средний балл
- Воронка конверсии (приглашены → начали → завершили)
- Распределение баллов кандидатов
- AI-рекомендации (Рекомендовано, Хороший кандидат, Требует внимания)

#### 4. **Управление кандидатами**
- Просмотр списка кандидатов
- Оценки и статусы (Завершено, В процессе)
- Детальные профили кандидатов

### Для Кандидатов (Соискателей)

#### 1. **Прохождение AI-интервью**
- Получение приглашения от работодателя
- Прохождение интервью в удобное время (24/7)
- Запись видео-ответов на вопросы
- Возможность переписать ответ

#### 2. **Личный кабинет**
- Дашборд с общей статистикой:
  - Количество пройденных интервью
  - Средний балл
  - Динамика улучшения
- Последние интервью с результатами
- Рекомендованные вакансии

#### 3. **Поиск компаний и вакансий**
- Каталог компаний с фильтрацией
- Информация о компаниях:
  - Индустрия
  - Локация
  - Количество сотрудников
  - Открытые вакансии
- Сохранение избранных компаний

#### 4. **Отчеты по интервью**
- Детальный анализ каждого ответа
- Оценка речи и интонации
- Оценка профессиональных навыков
- Рекомендации по улучшению

#### 5. **Практика**
- Тренировочные интервью
- Подготовка к реальным собеседованиям

## Пользовательские сценарии

### Сценарий 1: HR создает интервью
1. HR заходит в портал
2. Переходит в раздел "Интервью"
3. Нажимает "Создать интервью"
4. Настраивает вопросы и критерии
5. Получает ссылку для кандидатов
6. Отправляет приглашения

### Сценарий 2: Кандидат проходит интервью
1. Получает email с приглашением
2. Переходит по ссылке
3. Регистрируется/входит в систему
4. Читает вопросы
5. Записывает видео-ответы
6. Отправляет интервью
7. Получает результаты и обратную связь

### Сценарий 3: HR анализирует результаты
1. Заходит в дашборд
2. Видит список кандидатов с баллами
3. Фильтрует по AI-рекомендациям
4. Открывает детальный отчет
5. Принимает решение о дальнейших шагах

## Примечания для ИИ-ассистента

При работе с этим проектом учитывайте:

1. **Консистентность дизайна**: Все новые компоненты должны следовать существующей дизайн-системе (dark theme, purple accents, glassmorphism)

2. **Типизация**: Используйте TypeScript для всех новых компонентов

3. **Responsive design**: Все компоненты должны быть адаптивными

4. **Accessibility**: Добавляйте ARIA labels и обеспечивайте keyboard navigation

5. **Performance**: Используйте Server Components где возможно, оптимизируйте изображения

6. **Naming conventions**: 
   - Компоненты: PascalCase
   - Файлы: camelCase или kebab-case
   - CSS классы: Tailwind utilities

7. **Code style**:
   - Используйте functional components
   - Hooks для state management
   - "use client" для client components
   - Destructuring для props

8. **Локализация**: Весь текст на русском языке

9. **Иконки**: Используйте Lucide React для всех иконок

10. **Анимации**: Используйте Framer Motion для сложных анимаций, CSS transitions для простых
